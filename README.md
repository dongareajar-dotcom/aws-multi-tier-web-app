# AWS Multi-Tier Web Application

## Architecture
- Frontend: EC2 + Nginx
- Backend: EC2 + FastAPI
- Load Balancer: AWS ALB
- Network: VPC, Subnets, Security Groups

## Features
- Scalable architecture
- Load balanced frontend
- REST API backend
- Cloud deployment on AWS

## Tech Stack
- AWS EC2
- AWS ALB
- VPC
- FastAPI
- HTML + JavaScript
